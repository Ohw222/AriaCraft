### 2. Installation
Installez Skript et tous les plugins cités ci-dessus, reloadez ou redemarrez votre serveur, puis mettez les sk dans
<br>		-> plugins/Skript/scripts
<br>Et reloadez Skript avec la commande "/sk reload all"
